# LinuxDo Side Reader

一个 Chrome 浏览器扩展，在 [linux.do](https://linux.do) 中以右侧滑出面板方式打开帖子，告别页面跳转。

## ✨ 功能特性

- 🖱️ **点击即预览** — 点击帖子链接，在右侧滑出面板中打开，无需跳转页面
- ⚡ **JSON 极速预览** — 点击后先用 `/t/{id}.json` 立即展示正文，完整页面后台接管
- 🔥 **SPA 预热** — 空闲时预热 Discourse 壳，优先客户端路由切帖，减少冷启动
- ♻️ **同帖秒开** — 关闭面板后保留已加载内容，再次打开同一帖无需重载
- ↔️ **拖拽调宽** — 拖拽面板左边缘可自由调整面板宽度
- 🔗 **新标签页打开** — 点击头部按钮可在新标签页中打开当前帖子
- ⌨️ **Esc 关闭** — 按 Esc 键或点击遮罩区域关闭面板
- 🎨 **深色模式** — 自动适配 Discourse 深色主题
- 📱 **响应式** — 面板宽度自适应，最小 350px，最大 100% 视口宽度

## 📦 安装方式

### 开发者模式安装（推荐）

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入扩展管理页面：`chrome://extensions/`
3. 开启右上角的 **「开发者模式」** 开关
4. 点击 **「加载已解压的扩展程序」** 按钮
5. 选择本项目的文件夹（包含 `manifest.json` 的目录）
6. 扩展安装完成，访问 [linux.do](https://linux.do) 即可使用

### 生成图标（可选）

如需自定义图标，可运行：
```bash
node generate-icons.js
```
或在浏览器中打开 `icons/generate.html` 手动下载。

## 🚀 使用方法

1. 安装扩展后，打开 [linux.do](https://linux.do)
2. 在帖子列表中点击任意帖子标题
3. 帖子内容将在右侧滑出面板中加载显示
4. 可通过以下方式关闭面板：
   - 点击面板右上角的 **✕** 按钮
   - 按 **Esc** 键
   - 点击左侧半透明遮罩区域

### 快捷操作

| 操作 | 说明 |
|------|------|
| 单击帖子 | 在右侧面板打开 |
| Ctrl/⌘ + 单击 | 在新标签页打开（保留原生行为） |
| Esc | 关闭面板 |
| 拖拽左边缘 | 调整面板宽度 |
| 点击遮罩 | 关闭面板 |

## 📁 项目结构

```
liunxdo/
├── manifest.json        # Chrome 扩展配置文件 (Manifest V3)
├── content.js           # 内容脚本 - 拦截链接、管理面板
├── styles.css           # 面板样式（含深色模式适配）
├── generate-icons.js    # 图标生成脚本
├── icons/
│   ├── icon16.png       # 16x16 图标
│   ├── icon48.png       # 48x48 图标
│   ├── icon128.png      # 128x128 图标
│   └── generate.html    # 浏览器端图标生成器
└── README.md            # 说明文档
```

## 🔧 技术说明

- 使用 **Manifest V3** 规范
- 通过事件委托拦截 `/t/{slug}/{id}` 格式的链接点击
- 使用 **持久预热 iframe** 加载完整 Discourse，优先客户端路由切帖
- 点击后先用 JSON 渲染极速预览层，iframe 就绪后无缝切换到完整页
- 悬停 / pointerdown 预取 JSON，关闭后保留 SPA 壳供下次复用
- CSS 过渡动画实现平滑的滑入/滑出效果
- 支持拖拽调整面板宽度
