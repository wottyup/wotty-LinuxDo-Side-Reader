# LinuxDo Side Reader

一个 Chrome 浏览器扩展，把 [linux.do](https://linux.do) 改成**双栏视图**：左半边帖子列表，右半边当前帖子页面，无需来回跳转。

## ✨ 功能特性

- 🪟 **全站双栏** — 在 linux.do 上自动把页面拆成左右两栏
- 📋 **列表页** — 当前列表放左半，右半用 iframe 显示被点击的帖子
- 📖 **帖子页** — 当前帖子放右半，左半用 iframe 加载 `/latest` 列表
- 🔥 **悬停预热** — 列表页悬停 / 触摸时即时导航 iframe 到目标帖子，点击后更快打开
- ♻️ **同帖秒开** — 已加载的帖子在右栏可复用
- ↔️ **拖拽调比例** — 拖动中间分隔条可自由调整两栏比例（20%–80%）
- 🔗 **新标签页打开** — 点击头部按钮在新标签页中打开当前栏内容
- ⬅️ **折叠双栏** — 点击折叠按钮或按 Esc 收起双栏恢复整页，再次展开恢复
- 🎨 **深色模式** — 自动适配 Discourse 深色主题

## 📦 安装方式

### 开发者模式安装（推荐）

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入扩展管理页面：`chrome://extensions/`
3. 开启右上角的 **「开发者模式」** 开关
4. 点击 **「加载已解压的扩展程序」** 按钮
5. 选择本项目的文件夹（包含 `manifest.json` 的目录）
6. 扩展安装完成，访问 [linux.do](https://linux.do) 即可使用

### 生成图标（可选）

如需自定义图标，可运行：
```bash
node generate-icons.js
```
或在浏览器中打开 `icons/generate.html` 手动下载。

## 🚀 使用方法

1. 安装扩展后，打开 [linux.do](https://linux.do)
2. **列表页**（如 `/latest`）：页面自动变成左半列表 + 右半占位，点击任一帖子，帖子在右半加载
3. **帖子页**（如 `/t/.../123`）：页面自动变成右半当前帖子 + 左半 `/latest` 列表，在左半点击其他帖子可直接切换右半内容
4. 可通过以下方式操作双栏：
   - 拖动中间**分隔条**调整两栏比例
   - 点击头部 **⇥** 按钮 **折叠**双栏（恢复整页）
   - 折叠后点击角落的 **⇤** 按钮**展开**恢复双栏
   - 按 **Esc** 键折叠双栏

### 快捷操作

| 操作 | 说明 |
|------|------|
| 单击帖子（列表页） | 在右栏打开 |
| 单击帖子（左栏 /latest） | 切换右栏到该帖子 |
| Ctrl/⌘ + 单击 | 在新标签页打开（保留原生行为） |
| Esc | 折叠双栏 |
| 拖拽分隔条 | 调整两栏比例 |

## 🔧 技术说明

- 使用 **Manifest V3** 规范
- 通过事件委托拦截 `/t/{slug}/{id}` 格式的链接点击
- **不移动 Discourse DOM**，只用 CSS 收窄当前页 + `position:fixed` iframe 盖住另一半
- 列表页：当前页放左半，右半 iframe 显示帖子；悬停 / 触摸即导航预热
- 帖子页：当前页放右半，左半 iframe 加载 `/latest`，拦截 iframe 内 `/t/` 点击并用顶层 Discourse 路由切换右半
- 优先走 Discourse 客户端路由切帖，切换帖子时尽量不冷启动
- 支持拖拽分隔条调整两栏比例（持久化），可折叠恢复整页
